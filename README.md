# 🏦 Bank Management System

A simple console-based banking system written in **C#**.  
This project demonstrates fundamental Object-Oriented Programming concepts and basic data management using a `Dictionary`.

---

## 🚀 Features

- ✅ Create a new bank account  
- 💰 Deposit money  
- 💸 Withdraw money  
- 📊 Check account balance  
- 🌍 Multi-language support (Bulgarian / English)  
- 🧠 Basic input validation and error handling  

---

## 🛠 Technologies Used

- C#
- .NET
- Console Application
- Dictionary for data storage
- Object-Oriented Programming principles
